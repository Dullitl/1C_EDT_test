Настройка учета зарплаты и управления персоналом

В форме Настройка параметров учета некоторые настройки указываются раздельно по Организациям. Для некоторых настроек нужно дополнительно указать дату (месяц), начиная с которой действует настройка, или год, на который она устанавливается. Программа сохраняет историю изменения таких настроек.

Закладка "Алгоритмы расчетов"

На закладке Алгоритмы расчетов указываются некоторые особенности алгоритмов расчетов.
 
Первая особенность касается удержания за неотработанный отпуск при увольнении. В соответствии с Трудовым кодексом РФ работодатель имеет право на удержание за неотработанные сотрудником дни отпуска (например, в том случае, если сотрудник "отгулял" отпуск авансом и увольняется). В зависимости от установки переключателя Удержание за неотработанный отпуск при увольнении удержанные суммы будут рассматриваться программой:

  • как удержание, не уменьшающее налоговые базы по НДФЛ и ЕСН,
  • как корректировку ранее сделанного начисления, уменьшающую налоговые базы по НДФЛ и ЕСН.

Вторая особенность касается пересчета месячного оклада в часовую ставку для расчета оплаты сверхурочных часов.  С помощью переключателя При пересчете месячного оклада в часовую ставку использовать можно выбрать один из следующих вариантов:

  • Месячная норма времени по графику сотрудника – в этом варианте при пересчете будет использоваться среднемесячное количество часов с учетом длительности рабочей недели сотрудника.
  • Среднее за год количество часов в месяце – при пересчете будет использоваться месячная норма времени в часах по графику сотрудника; этот порядок должен быть закреплен в коллективном договоре.
  • Месячная норма по производственному календарю - в этом варианте при пересчете будет использоваться среднемесячное количество часов с учетом длительности рабочей недели по производственному календарю в соответствии с Приказом Минздравсоцразвития России от 13.08.2009 № 588н.

На закладке также имеется возможность управления результатом расчета количества дней компенсации отпуска при увольнении. Если установить флажок Округлять дни компенсации отпуска при увольнении, то округление будет проводиться до целых дней.

Закладка "Кадровый учет"


На закладке Кадровый учет указываются особенности кадрового учета раздельно по Организациям.
 
Если установить флажок Контролировать штатное расписание при кадровых изменениях, то при вводе новых кадровых документов программа будет проверять их соответствие штатному расписанию.

Если установить флажок Единый нумератор кадровых документов, то будет применяться сквозная нумерация при вводе документов Прием на работу, Кадровое перемещение, Увольнение и аналогичных.

Если установить флажок Показывать табельные номера в документах, то в формах документов будет отображаться колонка с табельными номерами сотрудников.

Если установить флажок Показывать табельные номера в документах, то в формах документов будет отображаться колонка с табельными номерами сотрудников.

Переключатель "Остатки отпусков будут уменьшаться при регистрации фактических отпусков" указывает какими документами будет производится списание фактических отпусков кадровыми или расчетными.

Закладка "Расчет зарплаты"

На закладке Расчет зарплаты указываются особенности расчета зарплаты раздельно по Организациям.

Если установить флажок Контролировать пересечение периодов начислений, регистрирующих норму времени, то программа будет контролировать факт двойной оплаты одного и того же периода времени.

Если установить флажок Взаиморасчеты по зарплате ведутся в разрезе месяцев ее начисления, то программа будет учитывать задолженности по зарплате с детализацией по месяцам (в противном случае – только общей суммой).

Если установить флажок При начислении НДФЛ принимать исчисленный налог к учету как удержанный, то начисленный НДФЛ при проведении документов сразу же будет регистрироваться как удержанный. Такой подход упрощает формирование отчетности в Организации, где заработная плата выплачивается регулярно и без задержек.

Флажок Расчет зарплаты по ответственным используется в ситуации, когда за расчет зарплаты по подразделениям отвечают расчетчики. Это позволяет каждому расчетчику работать только с сотрудниками, которые закреплены за ним. При установленном флажке документ начисления заплаты будет заполняться списком сотрудников, закрепленным за расчетчиком, если предварительно в поле Ответственный документа указать расчетчика.

Закладка "Плановый аванс"

На закладке Плановый аванс указываются особенности начисления аванса по Организациям.

Если установить флажок Учитывать невыход, то при формировании списка на получение планового аванса будут исключены сотрудники, у которых на расчетную дату планового аванса не было зарегистрировано отработанных дней.

Закладка "Расчет НДФЛ"

На закладке Расчет НДФЛ задается порядок применения стандартных налоговых вычетов при исчислении НДФЛ на очередной налоговый период. Предусмотрено два варианта:

  • Стандартные вычеты применяются нарастающим итогом в течение налогового периода – в этом случае вычеты, на которые имеет право налогоплательщик (сотрудник) с начала года по месяц исчисления налога, применяются к налоговой базе, исчисленной нарастающим итогом за год,
  • Стандартные вычеты применяются в пределах месячного дохода налогоплательщика – в этом случае вычеты, на которые имеет право налогоплательщик (сотрудник) в каждом месяце налогового периода, применяются к налоговой базе, исчисленной за этот месяц (вариант соответствует положениям письма Минфина РФ от 7 октября 2004 г. №03-05-01-04/41).

Допускается изменять порядок применения стандартных налоговых вычетов в течение года, после этого при исчислении НДФЛ за очередной месяц налогового периода будут пересчитаны суммы предоставленных вычетов, а также суммы налога за предыдущие месяцы.

Закладка "Вычеты по НДФЛ"

В списке Размеры вычетов по НДФЛ показываются текущие размеры налоговых вычетов.

В списке Вычеты к доходам по НДФЛ показываются текущие размеры налоговых вычетов к доходам.

Закладка "Размеры пособий"

На закладке Размеры пособий указываются размеры государственных пособий.

Закладка "ФСС НС и ПЗ"

На закладке ФСС НС и ПЗ раздельно по Организациям указывается страховой тариф на обязательное страхование от несчастных случаев на производстве и профессиональных заболеваний.

Закладка "Прочие ставки"

На закладке Прочие ставки указываются вспомогательные данные, которые могут использоваться программой:

  • Ставка рефинансирования ЦБ РФ
  • Минимальный размер оплаты труда.

Закладка "Расчет ЕСН"

На закладке Расчет ЕСН (Единого социального налога) следует указать точность исчисления ЕСН) и взносов в Пенсионный фонд России (ПФР) раздельно по Организациям. Доступны следующие градации точности:

  • С максимальной точностью - до долей копеек,
  • В рублях и копейках,
  • В рублях.

Кроме того, следует указать год, начиная с которого применяются настройки.
