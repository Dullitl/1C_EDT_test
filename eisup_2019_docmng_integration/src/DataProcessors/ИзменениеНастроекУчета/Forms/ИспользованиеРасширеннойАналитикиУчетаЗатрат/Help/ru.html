Настройка учета «Использовать расширенную аналитику учета затрат»

Назначение

Включение данной настройки позволяет использовать подсистему расширенной аналитики учета себестоимости МПЗ и затрат.

Преимущества подсистемы расширенной аналитики:

  • Использование для учета всех затрат и себестоимости МПЗ единого регистра накопления с минимальным количеством измерений
  • Ведение учета всех затрат с детализацией до номенклатуры, характеристики и серии
  • Совмещенный учет данных бухгалтерского и налогового учета в одном регистре накопления
  • Применение метода решения системы линейных уравнений при расчете себестоимости и корректировке движений МПЗ
  • Движения по регистрам учета формируются всегда при проведении документов
  • Не требуется выполнение процедуры проведения по партиям
  • Не требуется соблюдение последовательности и даты ввода приходных и расходных документов

 

В подсистеме расширенной аналитики учета используются новые регистры накопления:

  • Учет затрат - для целей управленческого учета
  • Учет затрат (бухгалтерский и налоговый учет) - для целей регламентированного учета

Порядок работы при включении настройки

Выбор режима ведения учета

Доступны 3 режима работы с подсистемой:

1. Регламентированный учет

2. Регламентированный учет с дополнительной аналитикой

3. Управленческий и регламентированный учет

Каждый режим определяет соотношение регламентированного и управленческого учетов при ведении учета затрат на предприятии, а также определяет состав аналитических разрезов учета затрат.

Настройка детализации затрат

Доступна только для режимов «Регламентированный учет с дополнительной аналитикой» и «Управленческий и регламентированный учет». При этом можно настраивать только некоторые разрезы аналитики учета затрат. Флажки, соответствующие другим разрезам - недоступны.

  • Для режима «Регламентированный учет с дополнительной аналитикой» можно настраивать детализацию затрат только по регл. учету
  • Для режима «Управленческий и регламентированный учет» - только по упр. учету

 

Формирование документов ввода остатков

Включение настройки может происходить в одном из двух режимов:

  • Полный
  • Выборочный

Работа в полном режиме

В полном режиме все изменения выполняются за одну итерацию:

  • заполнение регистров расширенной аналитики
  • установка значения настройки учета

Необходимо установить флаг "Использовать расширенную аналитику учета затрат", указать дату начала использования расширенной аналитики и нажать кнопку "Изменить настройку".

Перед формированием документов ввода остатков существующие на указанную дату документы ввода остатков будут помечены на удаление 

Работа в выборочном режиме

Данный режим полезен при большом объеме данных информационной базы. Можно разделить процесс заполнения регистров расширенной аналитики на несколько этапов (по разделам учета, видам учета).

Порядок работы:

  1. Указать дату на которую необходимо выполнить заполнение регистров расширенной аналитики: дату расчета
  2. Выполнить заполнение регистров расширенной аналитики. Данное действие может выполняться поэтапно
    1. Настроить исходные данные для заполнения регистров с помощью таблицы настроек. Уже рассчитанные разделы и разделы, по которым не требуется формировать остатки, выделяются в таблице бледно-голубым цветом фона
    2. Нажать кнопку "Рассчитать", дождаться окончания процесса расчета
    3. Повторять операцию до тех пор, пока не будет выполнено заполнение для всех исходных данных
    4. Уже рассчитанные данные можно увидеть в таблице "Сформированные остатки"
      1. можно установить отбор по разделу учета, на котором установлен курсор в таблице настроек
      2. при необходимости остатки могут быть удалены (кнопка "Удалить"), и после этого расчет можно выполнить повторно
      3. по двойному щелчку можно открыть соответствующий документ ввода остатков
  3. После того, как все исходные данные будут обработаны, нажать кнопку "Далее". В открывшемся окне установить флаг "Использовать расширенную аналитику учета затрат" и нажать кнопку "Изменить настройку"
  4. При необходимости можно установить значение настройки учета до окончания заполнения регистров, нажав кнопку "Далее" и затем воспользоваться кнопкой "Изменить настройку", но этой возможностью следует пользоваться осознанно.

Необходимо иметь в виду, что после начала заполнения регистров расширенной аналитики не следует изменять документы, введенные датой меньшей или равной дате расчета. В противном случае заполнение может оказаться некорректным.

Выполняемые в процессе включения настройки операции отражаются в журнале регистрации

Порядок работы при выключении настройки

  1. Отключить признак использования расширенной аналитики
  2. Установить значение настройки учета (кнопка "Изменить настройку")

При выключении настройки данные регистров расширенной аналитики автоматически не очищаются.
