Набор персонала

Основная (нижняя) область формы обработки предназначена для работы с заявками кандидатов и носит название Активные заявки кандидатов. В полях вспомогательной (верхней) области Неразобранные письма отображается информация о письмах, пришедших по электронной почте на адреса с назначением Переписка с кандидатами. Если работа с электронной почтой не ведется, область с неразобранными сообщениями можно скрыть.

Активные заявки кандидатов

Основная область формы Активные заявки кандидатов состоит из нескольких частей:

  • В левой части отображается дерево заявок кандидатов, отсортированных и сгруппированных по реквизиту заявки. Реквизит, по которому группируется и сортируется список, выбирается пользователем. Такими реквизитами могут быть: дата события, состояние, наименование, вакансия, ответственный, подразделение, должность и группа заявок.
  • В правой части отображаются описание выбранной заявки (поле Описание заявки), список писем по заявке (верхняя часть поля Переписка по заявке) и текст выбранного электронного письма (нижняя часть поля Переписка по заявке). В поле Переписка по заявке отображаются письма по всем электронным адресам, а не только по адресам с назначением Переписка с кандидатами, как в поле вспомогательной области Не разобранные письма.

Менять реквизиты заявки можно непосредственно из описания заявки с помощью гиперссылок, вызывающих нужные формы, а также с помощью команд меню Заявки и контекстного меню.

Кроме изменения реквизитов заявки доступны такие функции как:

  • Переписка по заявке
  • Создание физического лица и сотрудника по заявке
  • Закрытие заявки

Неразобранные письма

Область работы с неразобранными письмами состоит из двух частей:

  • В левой части отображается дерево списка писем, отсортированных и сгруппированных по по дате или по отправителю, — в зависимости от выбора пользователя;
  • В правой части отображается текст выбранного электронного письма.

На основании письма можно сразу оформить заявку кандидата, после чего письмо исчезнет из списка неразобранных и будет прикреплено к созданной заявке.

Выбранное письмо можно отметить как нежелательное. Такое письмо будет перенесено в папку Нежелательные.

С помощью кнопок командной панели формы обработки можно открыть справочники Вакансии и Источники информации, а также вызвать обработку Кадровое планирование.
