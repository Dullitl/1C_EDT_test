Обработка «Групповая печать и отправка по электронной почте пакета документов на реализацию

Обработка предназначена для печати пакета документов на реализацию и их отправки по электронной почте.

В пакет входят следующие документы:

- Титульный лист с наименованиями и адресами отправителя и получателя для отправки в конверте формата А4 (в электронное письмо не включается);

- Счет на оплату;

- Торг-12 и/или Акт (для связанного со счетом на оплату документа "Реализация товаров и услуг");

- Счет-фактура по реализации.

Первоочередным в пакете является счет на оплату.

 

Выполнение обработки запускается нажатием на кнопку "Печать": при этом создаются/печатаются печатные формы документов и, если установлен флаг "Создать электронные сообщения", создаются электронные сообщения с вложенными печатными формами в формате Excel.

Заполнение списка счетов, по которым формируются печатные формы пакета документов

Список счетов на оплату, по которым необходимо распечатать пакет документов, заполняется в табличной части "Счета". Он может быть заполнен как автоматически при нажатии на кнопку "Заполнить" над таблицей счетов с отбором по Организации и/или Контрагенту в случае их предварительного указания в шапке обработки, так и вручную пользователями.

Настройка выполнения обработки

В шапке обработки есть флаги настройки выполнения обработки при нажатии на кнопку "Печать":

- Печатать титульную страницу - если флаг установлен, будет формироваться печатная форма титульной страницы. Титульная страница только выводится на печать, но не попадает в электронное письмо, отправляемое контрагенту.

- Создать электронные сообщения - если флаг установлен, будут созданы электронные сообщения (по одному на каждого контрагента, по которым в табоичной части счетов на оплату есть счета и установлен флаг). Электронные версии печатных форм в формате Excel будут помещены в электронные сообщения в качестве вложений.

- Вывести сразу на принтер - если флаг установлен, то печатные формы будут отправлены сразу на принтер, и не будут показаны на экране.

- Выводить диалог печати - если флаг установлен, то перед печатью каждого листа будет выведен стандартный диалог печати операционной системы. Флаг доступен только при выводе на печать сразу на принтер.

- Показать печатные формы - если флаг установлен, то печатные формы будут показаны на экране. Флаг доступен только если документы не отправляются сразу на печать. Его снятие может быть необходимо, если документы отправляются только по электронной почте без вывода на печать.

 

В шапке обработки обязательно должен быть указан вид адреса, который будет использоваться для определения для каждого контрагента адреса , который будет распечатан как адрес получателя.

Отправка контрагентам электронных сообщений

Для возможности создания электронных сообщений в программе обязательно должна быть заведена учетная запись электронной почты.

При создании электронных сообщений будет использован Адрес электронной почты контрагента для обмена электронными документами.

Созданные сообщения будут добавлены в таблицу "Электронные сообщения". Для отправки сообщений, в данной табличной части необходимо установить флаги рядом с отправляемыми сообщениями и нажать на кнопку "Отправить".

Отправляются только сообщения, которые не были отправлены в текущем сеансе работы с обработкой ранее (отправленные выделяются цветом и отметкой в колонке "Отправлено"). Отправленные сообщения, которые необходимо отправить/переслать еще раз, можно открыть в списке документов "Электронное письмо" и отправить снова.

Электронные сообщения будут создаваться столько раз, сколько будет нажата кнопка "Печать" при установленном флаге "Создать электронные сообщения".
