Настройки формата выгрузки отчетов статистики

Форма предназначена для указания формата, в котором будет производиться выгрузка статистических отчетов в электронном виде.

Формат выгрузки должен соответствовать версии системы сбора статотчетности, используемой территориальным органом Росстата.

«Организация» - в этом поле указывается организация, для которой необходимо настроить формат выгрузки статистических отчетов.

После сохранения настройки, выгрузка всех статистических отчетов по указанной организации будут производиться в выбранном формате.
