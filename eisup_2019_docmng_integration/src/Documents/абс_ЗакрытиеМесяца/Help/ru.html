Документ «Закрытие месяца»

Документ «Закрытие месяца» предназначен для постатейного закрытия затратный счетов группы 20, 26, 44.

  • бухгалтерскому учету;
  • налоговому учету.

За один период (месяц) документов может быть несколько, каждый из которых закрывает свои затратные счета по бухгалтерскому или налоговому учету.

Дата документа определяет период закрытия затратыных счетов.

Отражение в бухгалтерском/налоговом учете закрытия 20-х счетов

При проведении документа с установленным флажком БУ напротив пункта  «Закрытие счета 20» будет рассчитаны суммы, прошедшие в течение месяца  по 20-ым счетам в бухгалтерском/налоговом учете и сформированы проводки на счет 90.02.1 «Себестоимость продаж (по деятельности, не облагаемой ЕНВД)» с затратного счета  группы 20-х счетов».

Отражение в бухгалтерском/налоговом учете закрытия 26-х счетов

При проведении документа с установленным флажком БУ напротив пункта  «Закрытие счета 26» будет рассчитаны суммы, прошедшие в течение месяца  по 26-ым счетам в бухгалтерском/налоговом учете и сформированы проводки на счет 90.08.1 «Управленческие расходы (по деятельности, не облагаемой ЕНВД)» с затратного счета  группы 26-х счетов».

Отражение в бухгалтерском/налоговом учете закрытия 44-х счетов

При проведении документа с установленным флажком БУ напротив пункта  «Закрытие счета 44» будет рассчитаны суммы, прошедшие в течение месяца  по 44-ым счетам в бухгалтерском/налоговом учете и сформированы проводки на счет 90.07.1 «Расходы на продажу (по деятельности, не облагаемой ЕНВД)».
