1.   Общее

Документ «Заявка на ТМЦ» предназначен для подачи заявки на получение ТМЦ в рамках проекта (проектов).

Каждая Заявка на ТМЦ, прежде чем будет отгружена, должна пройти согласование.

 

2.   Заполнение документа Заявка на ТМЦ

Новый документ Заявка на ТМЦ создается в списке документов Заявка на обеспечение ТМЦ по проекту типовым образом:

·        Нажатием на кнопку Insert на клавиатуре;

·        Нажатием на кнопку  в командной панели в форме списка Заявки на обеспечение ТМЦ по проекту.

 

Общий вид документа Заявка на ТМЦ представлен на следующем рисунке:

Для Заявки на ТМЦ необходимо указать Вид операции:

·        Списание на расходы – если ТМЦ будут использоваться в работах, осуществляемых собственными силами;

·        Передача подрядчику – если работы будет выполнять подрядная организация.

По умолчанию при создании нового документа Заявка на ТМЦ вид операции устанавливается «Списание на расходы».

Далее форме необходимо заполнить все основные реквизиты.

Организация – выбор из справочника Организаций.

Склад отправитель – выбор из справочника Склады (Места хранения).

Склад получатель – выбор из справочника Склады (Места хранения).

Ответственный РП – Ответственный руководитель проекта. Выбор из справочника Пользователи.

Дата плановой поставки – дата поставки ТМЦ.

Заказчик – подразделение заказчик проекта.

Статус – статус согласования данной Заявки на ТМЦ. По умолчанию при создании нового документа устанавливается значение «Подготовка».

Далее на закладке «Аналитика» необходимо заполнить аналитики:

·        ЦФО – выбор из справочника «Подразделения»;

·        Контрагент – выбора из справочника «Контрагенты». Заполняется только в том случае, если вид операции документа Заявка на ТМЦ = Передача подрядчику.

·        ЦФУ – выбор из справочника «Центры финансового учета».

Далее необходимо указать проект или проекты, в рамках которых заказывается ТМЦ.

Чтобы задать несколько проектов, необходимо установить переключатель «Редактировать» из положения Без разбиения в положение Списком.

Станет доступной табличная часть «Доли распределения по проектам», в которой необходимо указать перечень проектов, на которые заказывается ТМЦ, а так же долю распределения заказываемых ТМЦ по этим проектам. Сумма долей распределения должна равняться 1.

После того, как будут заполнены аналитики, указаны проекты и распределены доли по проектам, на закладке «Оборудование» необходимо заполнить перечень заказываемых ТМЦ:

В табличную часть «Оборудование» ТМЦ можно добавить следующими способами:

·        Добавить сроку, нажав на клавиатуре кнопку Insert;

·        Добавить строку, на кнопку  в командной панели табличной части «Оборудование»;

·        Добавить сразу несколько ТМЦ. Для этого надо нажать кнопку «Подбор» на командной панели табличной части «Оборудование».

Для каждой позиции необходимо указать количество.

Если позиция заказывается для одного проекта, то в колонке «Проект» указывается этот проект.

Если для позиции не указан проект, а сама заявка сформирована сразу для нескольких проектов, то такое ТМЦ распределяется по проектам в соответствии с долей распределения, указанной на закладке «Аналитика».

Если заказываемая позиция должна быть распределена по проектам не стандартно, то для такого распределения используется инструмент «Расписать строку».

В специальной таблице распределения необходимо указать проекты, на которые распределяется ТМЦ, а так же количественный состав, распределяемый на каждый проект.

Кнопка «Распределить» предназначена также для распределения ТМЦ по проектам.

После того, как документ будет полностью заполнен, его необходимо запустить на согласование.

 

3.   Согласование Заявки на ТМЦ

После того, как все необходимые данные заполнены, документ Заявка на ТМЦ должен пройти цепочку согласования:

·        Подготовка;

·        Согласование РП;

·        Комплектация;

·        Скомплектовано.
