Общее описание документа

Документ "Настройка соответствия номенклатуры" служит для настройки соответствия использования номенклатуры к автоматизированным системам расчета (АСР).

Для просмотра списка документов "Настройка соответствия номенклатуры" в полном интерфейсе меню "Документы" раздела "Управление биллингом" следует выполнить команду "Настройка соответствия номенклатуры". На экран выводится список документов. Для удобства просмотра и поиска нужных документов с помощью дополнительных кнопок расположенных на панели можно быстро найти нужный документ.

Для создания документа используйте команду "Создать", для редактирования - "Изменить".

В документе используются следующие основные данные:

  • Номер - порядковый номер документа;
  • Дата - дата и время создания документа;
  • Организация - используется для выбора организации для которой будет использоваться настройка соответствия. По умолчанию, при создании нового документа устанавливается организация к которой относиться пользователь;
  • Ответственный - пользователь создавший документ. По умолчанию, при создании нового документа устанавливается текущим пользователем работающим с программой. Изменения элемента не разрешены;
  • Статус - текущий статус документа. Используется для согласования документа;
  • Страница 1 (Соответствие биллинговых услуг) - предназначена для задания соответствия номенклатуры к услугам автоматизированных расетных систем (АСР);
  • Страница 2 (Соответствие бюджетным аналитикам) - предназначена для задания соответствия номенклатуры к бюджетным аналитикам;
  • Страница 3 (История согласования) - используется для просмотра истории согласования документа по различным статусам;
  • Комментарий - произвольная строка для занесения дополнительной информации по документу.

    • Как работать с документом

    Если документ был только что создан, то необходимо заполнить данный документ с помощью кнопки Заполнить расположенной на панели. При нажатии на данную кнопку будет произведено заполнение данных на Странице 1 (Соответствие биллинговых услуг) и Странице 2 (Соответствие бюджетным аналитикам).

    Для задания правильного соответствия номенклатуры к услугам автоматизированных расетных систем (АСР) необходимо отредактировать данные на Странице 1 (Соответствие биллинговых услуг) и Странице 2 (Соответствие бюджетным аналитикам).

    Редактирование документа доступно только в статусе "Подготовка".

    После редактирования документа и отправки его на дальнейшее согласование необходимо перевести статус документа из "Подготовка" в "Согласование ОФК".
