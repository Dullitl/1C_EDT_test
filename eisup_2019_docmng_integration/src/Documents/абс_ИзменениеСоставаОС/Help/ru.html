Документ предназначен для ввода в программу информации об изменяющемся (изменившемся) номенклатурном составе внеоборотных активов.

Табличная часть документа может быть автоматически заполнена текущим составом внеоборотного актива. Для этого необходимо при заполненной шапке документа нажать на кнопку "Заполнить текущим составом".

Каждая строка табличной части разбита на две подстроки: в первой содержится информация о текущих номенклатурных позициях (материалах и оборудовании) в составе внеоборотного актива; во второй вводится информация о добавляемых номенклатурных позициях. В случае, когда происходит замена одной номенклатуры на другую, в первой строке указывается заменяемая (текущая) номенклатура, во второй - заменяющая. В колонке "Количество текущее" отображается количество текущей номенклатуры в составе внеоборотного актива на момент ввода документа в программу. Данное количество заполняется при автоматическом заполнении табличной части и выводится в информативных целях. В ячейке "Количество удалить" указывается количество текущей номенклатуры, исключаемой из состава внеоборотного актива. В ячейке "Количество добавить" указывается количество заменяющей номенклатуры. Если "Количество удалить" не заполнено, тогда текущая номенклатура не будет исключена из состава ОС. Аналогично - для ячейки "Количество добавить" по отношению к заменяющей номенклатуре.

Рекомендуемая последовательность действий при работе с документом:

1. Заполнение шапки документа.

2. Автоматическое заполнение табличной части нажатием на кнопку "Заполнить текущим составом".

3. Указание количества удаляемого для исключаемых номенклатур.

4. Указание заменяющих номенклатур и количества добавляемого в соответствующих строках.

5. Проведение документа.

!!! Учет состава основных средств и объектов строительства до принятия к учету в качестве основных средств ведется только по количеству. Суммовой учет по составляющим внеоборотных активов не предусмотрен.
