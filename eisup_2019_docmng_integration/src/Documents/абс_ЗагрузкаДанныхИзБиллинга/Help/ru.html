Документ «Загрузка данных из биллинга»

Документ предназначен для загрузки данных, выгруженных из биллинговых программ в формате ".xls", ".csv" и ".dbf". Загрузка производится в соответствии с правилами загрузки, выбираемыми из справочника "Правила загрузки пакетов данных". При выполнении данным документом загрузки могут создаваться документы: "Счет на оплату покупателю" (таб. часть "Услуги"), "Реализация товаров и услуг" (таб. часть "Услуги"), "Счет-фактура выданный".

Перед выполнением загрузки кроме правил необходимо также указать организацию, путь к файлу, из которого производится загрузка.

Загрузка выполняется в три этапа:

- чтение данных файла с загрузкой в табличную часть "Данные файла". Текстовые значения в данной табличной части могут быть отредактированы после загрузки; действие выполняется нажатием на кнопку "Прочитать данные из файла";

- сопоставление данных файла с данными базы данных по правилам загрузки; результат сопоставления будет отражен в табличной части "Загружаемые данные", которые также можно отредактировать вручную (например, указать обязательные реквизиты при неудачном сопоставлении, например, договоры); действие выполняется нажатием на кнопку "Сопоставить данные";

- создание документов (если установлены флаги создания для видов документов в строках таб. части "Загружаемые данные"); если документы в базе уже существуют (и указаны в таб. части), они будут перезаполнены и записаны; действие выполняется нажатием на кнопку "Создать документы".

Все три действия можно выполнить сразу нажатием на кнопку "Загрузить данные из файла" в верхней части формы. При этом создание документов не произведется при наличии ошибок.

 

Важно: При невозможности создания/перезаписи какого-то документа не сохранятся изменения ни в одном документе. Изменения, произведенные при сопоставлении данных (на втором этапе) при этом сохранятся.

При выполнении загрузки будут выдаваться сообщения об ошибках, если они будут. Также будет выдано сообщени о завершении выполнения загрузки (при любом результате).
