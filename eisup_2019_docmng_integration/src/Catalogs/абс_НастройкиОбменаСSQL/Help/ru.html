Заполнение Табличной части:

1.Колонка "Имя реквизита SQL"

Заполняется соответствующий реквизит в SQL таблице. К примеру. Есть таблица Kontr и у нее есть Поля в SQL "ID" ,"Name" . то  в колонке  "Имя реквизита SQL" необходимо просто указать "ID" для соответствующего

реквизита в SQL

2. Колонка "Путь к данным реквизита":

2.1 Заполнение соответствующего реквизита: просто необходимо указать соответствующий реквизит в метаданных 1С. к Примеру "НаименованиеПолное"

2.2 Заполнение реквизита по условию: В контексте условия есть переменная ТекОбъект с помощью которго вы можете вытащить что угодно. Пример:  ?(ТекОбъект .Код = "001",1,0).

 и Обязательно необходимо указывать  флажок "Путь к данным с условием";

2.2 Заполнение через ссылочный реквизит; В контексте условия есть переменная ТекОбъект с помощью которго вы можете вытащить что угодно. к примеру "ТекОбъект.ГоловнойКонтрагент.Код"

2.3 Заполнение из Регистра сведений "Версии объектов": В контексте условия есть переменная Запись,

 которая и является записью набора записей РС.ВерсииОбъектов. То есть к примеру хотим получить автора версии то пишем "Запись.АвторВерсии"
