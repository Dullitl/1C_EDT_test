Справочник "Правила загрузки пакетов данных"

Справочник предназначен для хранения информации о настройках загрузки пакетов данных из файлов ".xls", ".csv", ".dbf". В форме элемента указывается тип файла, номер строки файла, с которой начинается загрузка (начиная с 1), разделитель полей (для файлов ".csv"). Далее необходимо выбрать назначение загрузки, если производится, например, загрузка в табличную часть какого либо документа. Доступна загрузка в документы "Загрузка данных из биллинга". После указания назначения загрузки табличная часть "Правила загрузки объектов" будет заполнена доступными для загрузки реквизитами объекта выбранного вида в соответствии с выбранными пользователем в специально открывшейся форме. Далее для каждой строки необходимо указать настройки:

- Тип реквизита (если реквизит назначения составного типа);

- Значение для подстановки - это значение по умолчанию, которое будет подставляться при загрузке независимо от значений полей в загружаемом файле;

- Правило загрузки из справочника "Правила загрузки данных" - используется только для тех строк, в которых не установлено значение для подстановки, и в которых тип реквизита назначения (реквизита в базе данных) - ссылка на справочник или документ.

Примитивные типы (строка, число, дата, булево) загружаются без использования правил - приведением строки к соответствующему типу (если возможно). Для значений типа "Булево" возможны следующие строковые значения: числа (0-ложь, остальные-истина), строки ("Истина", "Ложь", "Да", "Нет", "True", "False"). Для значений типа "Дата" корректно отрабатываются строковые представления даты вида 'ггггММддЧЧммСС', "дд.ММ.гггг" и "дд.ММ.гггг 12:00:00".

В строках табличной части "Поля файла" необходимо указать номера колонок/полей файла, данных которых будут использоваться для соответствующего анализа. Например, если необходимо найти контрагента по ИНН и наименованию, и в файле поля ИНН и наименования контрагента - 1 и 5, соответственно, тогда

в элементе справочника правил загрузки данных для контрагента необходимо установить флаги поиска по ИНН и наименованию, а в правилах загрузки пакета данных в табличной части "Поля файла" с выделенной (активизированной) строкой реквизита контрагента в верхней табличной части для ИНН установить номер колонки 1, а для наименования 5.

В качестве правил могут указываться только правила загрузки справочников и документов.
