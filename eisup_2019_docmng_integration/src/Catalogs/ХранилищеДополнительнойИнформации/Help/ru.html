Справочник «Хранилище дополнительной информации»

Данный справочник предназначен для хранения файлов и изображений связанных с объектами базы данных, такими как:

  • Справочники:
    • Вакансии
    • Вопросы для анкетирования
    • Заявки кандидатов
    • Контрагенты
    • Компетенции сотрудников
    • Медицинские страховые полисы
    • Организации
    • Помещения
    • Территории
    • Физические лица

Все данные дополнительной информации делятся на 2 группы:

  • Изображения
  • Файлы

Изображения


При активизации строки с изображением, в правой половине показывается само изображение. При клике на картинку, она открывается увеличенная в отдельном окне.

Одно из изображений для некоторых объектов, например таких как элемент справочника «Номенклатура», можно установить в качестве основного. Поэтому на закладке «Изображения» в некоторых случаях появляется кнопка «Установить основное изображение».
