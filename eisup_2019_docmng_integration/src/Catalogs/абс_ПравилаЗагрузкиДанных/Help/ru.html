Справочник "Правила загрузки данных"

Справочник предназначен для хранения правил загрузки данных в разрезе каждого объекта базы данных. В шапке формы элемента выбирается тип объекта (справочник, табличная часть, регистр сведений или документ), выбирается вид объекта (справочника, табличной части, регистра или документа). Табличная часть настроек заполняется автоматически. В табличной части необходимо настроить правила сопоставления конкретного реквизита. Например, если необходимо найти элементы справочников по наименованию, тогда необходимо выбрать вид соответствующего справочника, в табличной части для реквизита "Наименование" флаг использования и флаг "Поле поиска", режим загрузки - "Искать".

Данный справочник используется для загрузки пакетов данных - для указания правил загрузки каждого вида данных из пакета в справочнике "Правила загрузки пакетов данных".

Для справочников, табличных частей и регистров сведений доступно с использованием данных правил создание новых/изменение существующих реквизитов (для невозможности создания новых элементов в форме наобходимо установить соответствующий флаг). Для документов доступен только поиск.

Номер первой строки данных табличного документа при использовании в пакетной загрузке передается из информации о пакете, значение в данном справочнике игнорируется.

Для работы в механизме загрузки пакетов данных можно использовать только правила загрузки в справочники, документы и табличные части.
