Данный отчет отбирает сведения о расходах предприятия, соответствующие строкам формы "1-предприятие",
и позволяет выгрузить их в формате, пригодном для загрузки в программу "Росстат. Обследование затрат предприятий".
На основании данных, загруженных в программу "Росстат. Обследование затрат предприятий", заполняются разделы формы ТЗВ-КСП.

 Для заполнения формы ТЗВ-КСП необходимы данные следующих строк формы "1-предприятие":
Для раздела 1 формы ТЗВ-КСП
• 75 "Расходы на приобретение сырья, материалов, покупных полуфабрикатов и комплектующих изделий для производства и продажи продукции (товаров, работ, услуг)";
• 79 "Расходы на приобретение топлива".
Для раздела 2 формы ТЗВ-КСП
• 132 "Прочие услуги производственного характера". Услуги, указанные в строках 133-136 формы "1-предприятие" исключаются пользователем вручную.
Для раздела 3 формы ТЗВ-КСП
• 154 "Прочие услуги".

 Списки счетов для отбора сведений настроены для типовой конфигурации и могут быть отредактированы пользователем.
Для редактирования нужно открыть панель пользовательских настроек, кнопка "Настройки". В панели доступны настройки
"Отбор" и "Редактирование параметров". Отредактированные настройки можно сохранить для дальнейшего применения,
используя кнопки командной панели "Сохранить настройки отчета", "Загрузить настройки отчета".
Для восстановления настроек по умолчанию служит кнопка пользовательской панели "Восстановить".

 Каждой строке отобранных сведений необходимо назначить код ОКПД из справочника кодов ОКПД.
Кнопка открытия справочника ОКПД расположена справа в командной панели.
При первом открытии справочника будет задан вопрос об автоматическом заполнении, из внешнего источника или из конфигурации.
При утвердительном ответе справочник будет заполнен.

 При нажатии клавиши "Enter" или двойном щелчке клавиши мыши в поле отчета "Код ОКПД" откроется заполненный справочник кодов ОКПД
для выбора соответствующего кода. Для упрощения подбора нужного кода в справочнике реализован полнотекстовый поиск, и подходящие коды
предлагаются сразу после открытия формы выбора. При первом обращении к справочнику запускается процедура обновления индекса
полнотекстового поиска, которая может выполняться достаточно долго.
 Каждый выбор сохраняется в регистре сведений и при следующем формировании отчета, подставляется в поле "Код ОКПД" автоматически.
Этот регистр сведений можно отредактировать, удалить старые записи. Кнопка открытия регистра сведений расположена справа в командной панели.

 После назначения кодов ОКПД в строках отчета, сведения выгружаются нажатием на кнопку "Выгрузить в файл".
Сформированный текстовый файл нужно загрузить в программу "Росстат. Обследование затрат предприятий" для дальнейшей обработки.
